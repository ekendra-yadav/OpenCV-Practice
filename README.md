# OpenCV Practice & Computer Vision Learning

This repository documents my journey learning Computer Vision using OpenCV and Python. It includes structured phases of learning and academic assignments.

## 🛠️ Tech Stack
* **Language:** Python
* **Library:** OpenCV (cv2)
* **Environment:** Virtual Environment (`cv_env`)

## 🚀 How to Run
1. Clone the repository.
2. Install dependencies: `pip install opencv-python`.
3. Run any script using: `python <filename>.py`.
